Catfe Diem

We are friendly, supportive and educational for users who’re moving into a new home. Intent on providing direct, organized information to reinforce user confidence and decision making.

UI Designer

Web Designer

Project

Wordpress Website

Timeline

3 weeks

Tools

Wordpress

Adobe Photoshop

Adobe Illustrator

Design Oppurtunity

Transform a restaurant brand concept into a fully functional, responsive website using WordPress.

Approach

I created “Catfe Diem,” a responsive website that works across all devices. It uses consistent branding, clear navigation, and optimized images for a smooth user experience.

Homepage

Clear homepage with consistent branding and simple navigation so users can quickly understand the café, and the services it offers

Adoption Page

This page helps users explore cats that are available for adoption. Users can quickly find information and understand each listing, by making the layout clear and easy to use,

Menu Page

This page displays the café’s food and drink menu in a structured layout. Also featured is a highlights section with popular items to draw attention to customer favourites and improve browsing.

Bookings

This page makes it easy for users to book a visit using a simple and clear form, as well as see important information.

This project explores the design of a cat café website focused on creating a warm, playful, and calming experience that reflects the in-person atmosphere of the café.

Design Intentions

01.

Create a cozy, welcoming digital atmosphere

02.

Evoke feelings of comfort, playfulness, and warmth through layout and imagery

03.

Make navigation for booking and visiting information clear and intuitive

04.

Give each cat personality to create emotional connections for visitors
